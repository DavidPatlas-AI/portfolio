  1. מכפילים את הספרות במקומות 0,2,4,6 ב-1
  2. מכפילים את הספרות במקומות 1,3,5,7 ב-2
  3. אם מתקבלת תוצאה דו-ספרתית, מחברים את שתי הספרות
  4. מחברים את כל התוצאות ומחשבים כמה צריך להוסיף כדי להגיע למספר שמתחלק ב-10
  5. ספרת הביקורת היא התוצאה של החישוב הזה (אם התוצאה היא 10, ספרת הביקורת היא 0)
