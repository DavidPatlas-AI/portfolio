מהו אלגוריתם בדיקת תקינות מספר רכב ישראלי?

בדיקת תקינות של מספר רכב ישראלי מבוססת על ספרת ביקורת שמחושבת באופן הבא:

  1. עבור פורמט חדש (8 ספרות): הספרה האחרונה היא ספרת ביקורת
  2. מכפילים את הספרות הנותרות במשקלים: 3,1,3,1,3,1,3
  3. מחברים את התוצאות ולוקחים את השארית בחלוקה ל-10
  4. ספרת הביקורת צריכה להשלים את הסכום למספר שמתחלק ב-10

עבור פורמט ישן (7 ספרות) משתמשים במשקלים: 3,1,3,1,3,1 ובודקים את הספרה השביעית.
