# ✡️ פרויקט תיקון השם ברוסית - מערכת דיגיטלית לזיכרון

מערכת מתקדמת לתיקון השמות הקדושים ברוסית עם ממשק רב-לשוני, מערכת תהילים דיגיטלית, סידור מלא ומערכת הנצחה אישית.

## 💝 הסיפור הרגשי מאחורי הפרויקט

בתור נכד של ניצולי שואה, תמיד שמעתי בבית על בעיית תיקון השמות הברוסית. סבתא שלי, שהיגרה מרוסיה, סיפרה לי איך השמות הקדושים נכתבו בצורה שגויה בתרגום, ואיך זה כואב לה שהיא לא יכולה לתקן את זה.

בשנתיים האחרונות החלטתי להקדיש את הכישורים הטכנולוגיים שלי לעזור למשפחות להנציח נכון את יקיריהם. **זה לא רק פרויקט טכנולוגי - זה שליחות רוחנית.**

כל פיצ'ר בפרויקט הזה נולד מסיפור אמיתי של משפחה שחיפשה דרך לכבד את זיכרון יקיריה.
